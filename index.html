Pascal Petivel :D

Bevezető

Azt hogy mi a program vagy programozás azt nem vagyok hajlandó belerakni. (kivéve ha megkér rá valaki)

A pascal nem case sensitive szóval ez: WriTeLN() ugyan az mint ez writeln(). De azért egy minimálisan figyeljünk odará hogy nézzen ki valahogy

A pascal programozáshoz célszerű letölteni valamit ammiben írni is tudjuk szóval itt van ez: Lazarus

Vagy van egy online alternatíva: online_pascal_compiler

Adat típusok

  • egész szám (integer): Pl.
    • 5
    • 3+4
  • szöveg (string): PL.
    • 'zsiráf'
    • 'nem zsiráf'
  • karakter (char): PL.
    • 'R'
    • 'ó'
    • 'k'
    • 'a'
  • törtszám (real): PL.
    • 2.006
    • 0.6
    • 0.1
  • logikai (boolean): PL.
    • TRUE
    • FALSE

Műveletek

  • Alap műveletek
    • +
    • -
    • *
    • /
    • >
    • <
    • =
    • <> nem egyenlő
  • szövegösszefűzés
    • 'záp'+'tojás' az eredménye 'záptojás'
  • egészek osztási maradéka
    • 5 mod 2 eredménye 1
  • egészek hányados
    • 25 div 7 eredménye 3
  • Logika
    • and
      • ( 0 = 0 ) and ( 10 < 99 ) eredménye TRUE
      • ( 0 = 10 ) and ( 10 < 99 ) eredménye FALSE
    • or
      • ( 0 = 0 ) or ( 10 < 99 ) eredménye TRUE
      • ( 0 = 10 ) or ( 10 < 99 ) eredménye TRUE
      • ( 0 = 10 ) or ( 10 < 1 ) eredménye FALSE
    • not
      • Ami eddig TRUE volt az most FALSE
      • not TRUE eredménye FALSE

Egyszerű program

  • A program neve
    • program project1;
  • Itt kezdődik a program futása
    • begin
  • Itt fejeződik be a futás
    • end.
  • Ki írja a benne lévő adatot a konzolra
    • Write('szöveg')

      Nem kezd új sort a kiírás után

    • Writeln('szöveg')

      Kezd új sort miután kiírta

  • A write és a writeln több argumentumot is el tud fogadni zeket vesszövel kell elválaszatni
  • Az argumentumokat egymás mellé fogja kiírni típustól függetlenül

Így néz ki a gyakorlatban egy egyszerű program

(*Ezek a jelek között lehet írni kommenteket*)

program project1;
begin               
    write('A pascal')  
    write('buta nyelv')
    readln;
end.

Változók

A változókban adatokat lehet tárolni

A változókkal úgy lehet műveleteket végezni mint ha csak az értékét írnánk oda

Teljesen ugyan az mint matek órán
  • Változó definiálása
    • Var változóNév : adatTípus;
    • Var kor : integer;
    • Var nev : string;
  • Értékadás változóknak
    • változóNév := adat;
    • kor := 18;
    • nev := 'Peti';

Adatbekérés változóba

    • Readln( változóNév )
    • Readln( nev )

Kód példa:

program valtozok;
Var nev:string;
BEGIN
    Write('Add meg a neved:');
    Readln(nev);
    Writeln('Szia, ' + nev + '!')
END.

Elágazás

Eléggé egyenes ez a dolog, ha teljesül az elágazás feltétele akkor lefut a benne lévő kód aztán fut tovább a kód
  • Egyszerű utasítás (ha igaz akkor csak egy sornak megfelelő kód tud lefutni utána)
    • If feltétel then Kód amit le akarsz futtatni;
    • If 0 < 21 then writeln('igen nulla kisebb mint 21');
  • Összetett utasítás
    • If feltétel then begin
      Kód amit le akarsz futtatni;
      end;
    • If "alma" <> "banán" then begin
      writeln('Nem');
      writeln('Az alma nem banán');
      end;

    Ha kezelni akarjuk azt is ha nem teljesült a feltétel

  • Egyszerű utasítás
      • If 0 < 21 then writeln('igen nulla kisebb mint 21') else Kód ami akkor fut le ha nem teljesült a feltétel
  • Összetett utasítás
    • If feltétel then begin
      Kód amit le akarsz futtatni;
      end else begin
      Kód ami akkor fut le ha nem teljesült a feltétel
      end;

Ciklusok

Ugyan úgy mint az IF-nél a begin és end; használatával ez is lehet több soros

Elöltesztelő ciklus

Megnézi hogy igaz-e a benne lévő feltétel, ha igen akkor kezd el futni ameddig igaz a feltétel

 
BEGIN
    while (TRUE) do writeln('nem szabadulsz tőlem');
END.

Hátultesztelős ciklus

Ugyan az mint az előző, annyi különbséggel, hogy legalább egyszer lefut a ciklus. (mivel "hátul" vagyis a ciklus végén teszteli, hogy igaz-e a feltétel)

 
Var i :integer;
BEGIN
    i := 0;
    repeat
        writeln(i);
        i := i + 1;
    until i = 10;
END.

Számlálós ciklus

Minden iterációnál hozzá ad egy megadott összeget egy változóhoz

 
var x :integer;
BEGIN
    for x:= 1 to 10 do writeln(x);
END.

Tömbök

A tömb is egy adat típus, kifejezetten gyakran használjuk.

A tömb adatok tárolására alkalmas.

A benne lévő adatokhoz a sorszámukkal (másnéven indexükkel) tudunk hozzáférni.
  • Lista hosszának megadása
    • változónév:array[hossz] of típus
    • lista:array[1..10] of integer
    • lista:array[1..3] of string
Var lista :array[1..10] of integer;
BEGIN
    lista[1] := 23;
    writeln('A lista első eleme:', lista[1]);
END.

Többdimenziós tömb

Nagyon hasonlít az előzőhöz, annyi hogy nem egy indexe van egy tömb elemének hanem kettő vagy több 

Var lista :array[1..5,1..3] of string;
BEGIN
    lista[1,1] := 'Szia';
    lista[1,2] := 'olvasó';
    writeln('A lista első eleme: ', lista[1,1], ' és ', lista[1,2]);
END.

Tömb bejárása

Var lista :array[1..10] of integer;
var x :integer;
BEGIN
    for x:= 1 to 10 do writeln(lista[x]);
END.

Eljárás és függvény

Mind a kettő egy kódrészlet amit meg tudsz hívni bármikor.

A kódban gyakranhasznált dolgokat célszerű a fentiek valamelyikébe rakni.

Az eljárás és a függvény közt a definiálásukon kívül annyi a különbség, hogy az eljárásnak nincs visszatérési értéke.

Eljárás

    Definiálása, létrehozása
  • Paraméter nélküli
    • Procedure eljárásNeve;
      begin
      kód
      end;
  • Paraméteres eljárás
    • Procedure(argumentumNév:argumentumTípus ) eljárásNeve;
      begin
      kód
      end; 
(*Paraméter nélküli*)

Procedure varj;
begin
    write('Nyomj entert a megoldáshoz');
    readln;
end;

BEGIN
    writeln('3+2=');
    varj;
    writeln('5.');
    writeln('5*7=');
    varj;
    writeln('35.');
END.


(*Paraméteres*)

Procedure csik(sz:integer);
var i:integer;
begin
    for i:=1 to sz do write('-');
    writeln;
end;

BEGIN
    csik(10);
    csik(20);
    csik(10);
END.
Function kob(a:real):real; Begin kob:=a*a*a; End;

Függvény

  • A visszatérési értéket úgy adjuk meg mintha értéket adnánk egy változónak aminek a neve megegyezik a függvény nevével
  • Definiálás
    • Function függvényNév(argumentumNév :argumentumTípus) :visszatérésiÉrtékTípusa;
      begin
      kód
      end;
  • Több argumentumot is meg lehet adni vesszővel elválasztva
    • Function függvényNév(argumentumNév :argumentumTípus, másikArgumentumNév :másikArgumentumTípus) :visszatérésiÉrtékTípusa; 
Function osszead(x,n:integer):integer;
begin
    osszead := x + n;
end;

BEGIN
    if osszead(2,3) = 5 then writeln('Működik!');
END.

Fájlkezelés

Szöveges fileok írása és olvasása
  • Fájl létrehozása:
    • Létre kell hoznunk egy text típusú változót:
      • változóNeve:text;
  • Hozzá kell rendelni a változóhoz egy adott fájlt annak útvonala segítségével:
    • assignfile(fájl, útvonal);
  • Megnyitás olvasásra:
    • A fájlt vagy írásra, vagy olvasásra nyitjuk meg:
      • reset(fájl);
  • Megnyitás írásra:
    • Ez új fájlt hoz létre, ha van már ilyen, azttörli:
      • rewrite(fájl);
  • Megnyitás írásra létező fájlnál úgy, hogy a végéhez fűzünk adatot:
    • append(fájl);
  • Fájl törlése:
    • erase(fájl);
  • Olvasás:
    • read(fájl, változók);
    • readln(fájl, változók);
  • Írás:
    • write(fájl, kifejezések);
    • writeln(fájl, kifejezések);
  • Vége van-e a fájlnak:
    • beolvastuk-e az utolsó adatot, függvény, igaz/hamis:
    • End Of File
      • eof(fájl);
  • Vége van-e egy sor beolvasásának:
    • eoln(fájl);
  • Fájl lezárása:
    • Erre főként írásnál van szükség, hogy az utolsó adatok is bekerüljenek a fájlba:
      • closefile(fájl);
  • A read és write utasításokat úgyanúgy kell használni, mint amikor a billentyűzetről olvasunk és a képernyőre írunk, csak most megadjuk a fájlt is.
